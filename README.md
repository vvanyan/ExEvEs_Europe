\# Title

Extreme Evaporation Events (ExEvEs) over Europe (1984–2023)



\# Objective

Understand and detect ExEvEs using the evapoRe and twc R packages.



\# Study area

Europe (Mediterranean, Western–Central, Northern regions)



\# Time period

1984–2023



\# References

1\. Markonis et al. (2025) — On the Definition of Extreme Evaporation Events, Geophysical Research Letters.  

2\. evapoRe: An R-based Application for Exploratory Data Analysis of Evapotranspiration.
